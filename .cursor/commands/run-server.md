# run-server

このプロジェクトのビジュアル開発に必要なサーバー群を一括起動するやで！😎✨

## 実行手順

以下のコマンドを順番に、あるいは別々のターミナルで実行してな！💪🔥

1. **OSCサーバーの起動**（WebSocket通信用）
   ```bash
   npm run osc-server
   ```

2. **開発サーバーの起動**（Vite）
   ```bash
   npm run dev
   ```

両方のサーバーが立ち上がったら、ブラウザで `http://localhost:3000` を開いて開発スタートや！🎉🤔
